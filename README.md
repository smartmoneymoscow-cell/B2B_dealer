# B2B Dealer Portal — Дилерский портал

Единая точка доступа для партнёров компании, позволяющая удалённо работать с ассортиментом, остатками, резервированием товара и документами без обращения к менеджеру по каждому вопросу.

## Платформы

| Канал | Описание |
|-------|----------|
| **Веб-интерфейс** | Работа через браузер |
| **Telegram Mini App** | Работа внутри Telegram |

Обе версии используют **единый backend** и работают с существующей инфраструктурой компании.

## Интеграции

| Система | Назначение |
|---------|------------|
| **1С** | Каталог товаров, цены, остатки, резервы, счета, статусы оплаты |
| **Bitrix24** | Авторизация, закреплённый менеджер, коммуникация |
| **Сайт компании** | Фотографии товаров, контактные данные, дополнительные материалы |

## Структура портала

### 1. Лицевая страница

Первый экран портала — контактная информация о компании и быстрый доступ к основным действиям.

- Логотип компании
- Контактные данные (адрес, телефон, e-mail)
- Кнопка «Перейти на сайт»
- Информация о закреплённом менеджере
- Кнопка «Перейти в каталог»

### 2. Каталог продукции

Основной рабочий раздел. Совмещает роль каталога товаров и прайс-листа.

**Возможности:**
- Поиск товаров
- Фильтрация по категориям
- Просмотр карточек товаров

**Карточка товара в каталоге:**
- Фотография
- Наименование
- Артикул
- Актуальная цена
- Остаток на складе
- Кнопка «Подробнее»

### 3. Карточка товара

Подробная информация о выбранной позиции.

**Основная информация:**
- Фотографии
- Наименование, артикул
- Описание и характеристики
- Цена
- Остаток на складе

**Материалы:**
- Инструкции
- Технические документы
- Сертификаты
- Каталоги продукции

**Создание резерва:**
Пользователь указывает количество и нажимает «Создать резерв». Запрос передаётся в 1С, пользователю отображается подтверждение и статус резерва.

### 4. Счета

Работа со счетами, сформированными по резервам или заказам.

- Номер, дата, сумма, статус оплаты
- Открытие счёта
- Скачивание счёта (PDF)
- Отслеживание изменения статуса после оплаты

### 5. Кабинет клиента

Информация о пользователе и настройки доступа.

- Данные пользователя
- Логин
- Смена пароля
- Выход из системы

## Главная задача

Предоставить дилеру актуальную информацию в режиме реального времени и сократить количество обращений к менеджерам за счёт самообслуживания.
